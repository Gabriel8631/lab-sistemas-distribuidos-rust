  Sistemas Distribuídos: Teoria e Prática em Rust

Bem-vindo ao repositório **rust-dist-systems-lab**! O objetivo deste projeto é documentar, estudar e implementar os principais conceitos e padrões arquiteturais que envolvem a computação distribuída, utilizando a linguagem Rust.

  Objetivo
Criar um laboratório prático para entender como construir sistemas escaláveis, resilientes e de alta disponibilidade.

 Estrutura do Laboratório

* **`/projects`**: Contém todas as provas de conceito e códigos em Rust.
  * `01-simple-rest-api`: Uma API base usando o framework Axum para demonstrar comunicação síncrona básica.

 Tecnologias
* **Rust** (Performance e Segurança de Memória)
* **Tokio** (Runtime Assíncrono)
* **Axum** (Framework Web)
